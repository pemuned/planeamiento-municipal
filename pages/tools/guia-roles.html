Guía de roles y responsabilidades

Imagine esta guía como las "reglas del juego" claras para el ETM. Es un documento sencillo y directo que define: quién es responsable de qué tarea, qué se espera de cada rol y cómo se van a tomar las decisiones importantes. Se utiliza al inicio del proyecto, una vez que el equipo está formado, para evitar confusiones, duplicidad de esfuerzos y conflictos. Su potencial es inmenso: asegura que todos sepan lo que tienen que hacer, que no haya tareas sin asignar y que las decisiones se tomen de forma eficiente. La oportunidad de mejora es que no sea un documento rígido; debe poder adaptarse si el equipo o el proyecto cambian.

¿En qué momento se utiliza?

Se utiliza al inicio del proceso, una vez que el ETM está conformado y se tiene una idea de las competencias disponibles, para formalizar las funciones y la forma de trabajo.

Ejemplos

  • Documento resumen con descripción de cargos, funciones y responsables.

Caso práctico (simulación)

En la Municipalidad de Morado, se realiza un taller en el que cada miembro escribe sus tareas esperadas y se construye colectivamente la guía final.

Inténtelo

  1. Identifique las tareas clave del proceso.
  2. Relacione las tareas con los roles posibles.
  3. Valide los roles y responsabilidades con el equipo.
  4. Difunda y aplique el documento.

Ejemplo de Guía de Roles y Responsabilidades – Equipo Técnico Municipal (ETM)

Contenido sugerido para la guía

  1. Introducción breve
    • Justificación del documento
    • Alcance y relación con el proceso del PDM
  2. Principios del trabajo en equipo
    • Colaboración
    • Transparencia
    • Responsabilidad compartida
    • Toma de decisiones basadas en evidencias
  3. Estructura del ETM
    • Nombre y perfil de cada integrante
    • Representación institucional o técnica
  4. Tabla de roles, tareas y responsabilidades (ver más abajo)
  5. Mecanismos de coordinación
    • Reuniones periódicas
    • Herramientas de comunicación
    • Resolución de conflictos
  6. Firma simbólica de compromiso (opcional)
    • Para reforzar la corresponsabilidad y legitimidad del equipo
