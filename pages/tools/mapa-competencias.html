Mapa de competencias del equipo técnico municipal

Esta herramienta es fundamental para la optimización del talento humano del ETM. Tiene como propósito identificar fortalezas, habilidades y conocimientos específicos de los integrantes del ETM. Sirve para identificar qué fortalezas y conocimientos aporta cada integrante.

¿En qué momento se utiliza?

Se utiliza en la fase de conformación o consolidación del equipo, o al inicio de un proyecto complejo, para hacer un inventario de las habilidades, conocimientos, experiencias y fortalezas de cada integrante. Su potencial reside en asignar roles de manera más efectiva, identificar brechas de conocimiento que requieran capacitación, y potenciar la colaboración al saber quién tiene experticia en qué área. La oportunidad de mejora es que no solo identifique competencias técnicas, sino también habilidades blandas (comunicación, liderazgo, resolución de conflictos) y se actualice periódicamente.

Ejemplo

  • Matriz de habilidades: Una tabla donde se listan las competencias clave para el PDM (ej. gestión de proyectos, contratación administrativa, participación ciudadana, comunicación, ordenamiento territorial) y cada integrante evalúa su nivel de experticia.
  • Mapa visual: Un esquema gráfico donde se representan las competencias de cada persona, quizás con colores o iconos.

Caso práctico (simulación)

En la Municipalidad de Río Celeste, los integrantes del ETM completan una ficha de habilidades técnicas, luego se cruzan los datos en una matriz visual para identificar liderazgos temáticos.

Inténtelo

  1. Identifique competencias clave: Defina las habilidades y conocimientos esenciales para el éxito del PDM (ej. planificación estratégica, finanzas municipales, impacto ambiental, comunicación, facilitación de grupos).
  2. Diseñe el formato: Crear una matriz, cuestionario o plantilla para registrar las competencias.
  3. Haga una autoevaluación o evaluación cruzada: Los miembros pueden autoevaluarse o, si es apropiado y con la debida sensibilidad, puede haber una evaluación por parte de un coordinador o pares.
  4. Recopile información: Distribuya el formato y recoja los datos de cada miembro.
  5. Analice el mapa: Identifique fortalezas colectivas, áreas con alta experticia y, crucialmente, las brechas de competencias donde el equipo puede necesitar apoyo externo o capacitación.
  6. Visualice los resultados: Cree un mapa visual claro que pueda ser compartido con el equipo (de manera agregada o individualmente, según la confidencialidad).
  7. Utilizar el mapa: Asigne tareas y roles en función de las fortalezas identificadas, diseñar planes de capacitación o buscar apoyo externo para las brechas.
  8. Revisión periódica: Actualice el mapa de competencias a medida que los miembros adquieren nuevas habilidades o se integran nuevos profesionales.
